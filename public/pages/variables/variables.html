Variables in JavaScript

In JavaScript you can use three different keywords to declare a variable:
- var
- let
- const

var

Using var is the old way of writing JavaScript variables and is generally not recommened today. When you declare a variable with var, the variable is global unless written in a function, which means that is accessible from anywhere in the code. Variables declared with var is also always reassignable.

let and const

Variables assigned with the let keyword are block scoped, which means that they are only accessible in the block or nested block which they are declared in. The difference between let and const is simply that variables declared with let are reassignable, while variables declared with const are not.

Example

To illustrate the difference between var and let, consider the following code: 

    
        {
            let a = 7;
        }
        console.log(a); // produces an error because a is not declared in this scope
        
        {
            var b = 7;
        }
        console.log(b); // prints 7
