What is Node.js

Node.js is a JavaScript runtime which means that you can run JavaScript locally without a browser. A common use for Node.js is to create and run servers written in JavaScript.

Setup and basics

First you need to download Node.js using this link. Choose the approriate download depending on your operating system and then click install and follow the onscreen instructions. To check if Node has been correctly installed, open a terminal and type node --version. If you see a version number, Node has been correctly installed.

To run a JavaScript file, open a terminal and type node [filename including extension]. If for example your file is called app.js, then you would enter the command node app.js.

Example "hello world" program to run:


        console.log("Hello world");

package.json

It is common to have a file called package.json in every Node project. package.json is a file formatted as JSON. The file can contain dependencies, in which case the dependencies will be installed when you type npm install in a terminal. npm is a package that was installed at the same time as Node.js. If you install a specific package via npm, the newly added dependency will also appear in package.json. package.json can also include metadata about for example the name of your package, version and other information. The file also contains the "main" property, which declares the main entry point of your package.
